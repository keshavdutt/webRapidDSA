Nearest Greatest Element to the Right

Keshav Last Updated : Apr 17, 2024 17:32 PM IST

The problem of finding the "Nearest Greatest Element to the Right" for each element in an array can be efficiently solved using a stack. Here's the JavaScript solution along with the time and space complexities:
Problem Statement
Given an array, for each element, find the nearest element to the right that is greater than the current element. If no such element exists, use -1.
Solution
We can solve this problem using a stack in linear time, ЁЭСВ ( ЁЭСЫ ) O(n).
Algorithm

  • Initialize an empty stack and an array result to store the nearest greater elements.
  • Traverse the array from right to left.
  • For each element:
  • Pop elements from the stack while the stack is not empty and the top element of the stack is less than or equal to the current element.
  • If the stack becomes empty, it means there is no greater element to the right, so add -1 to the result.
  • Otherwise, the top element of the stack is the nearest greater element to the right, so add it to the result.
  • Push the current element onto the stack.
  • Reverse the result array to maintain the order corresponding to the original array.
